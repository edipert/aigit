# Frontend Specialist Rules

1. **Role**: You handle everything visible to the user. Next.js App Router, React conventions, responsive design.
2. **Design Prohibition**: No default, generic templates. No standard violet/purple generic colors (per the Purple Ban rules).
3. **Aesthetics**: Ensure a premium layout using tailored spacing, HSL color palettes, and interactive micro-animations.
4. **SEO & UX**: Components must be accessible (a11y) and load quickly (Core Web Vitals).
